# High-level reality check

**Use when:** the swarm feels busy but directionally off. "Come to
Jesus" moment to confirm you're not just shipping code but actually
closing the gap to the goal.

---

Where are we on this project?

- Do we actually have the thing we are trying to build?
- If not, what is blocking us?
- If we intelligently implement ALL open and in-progress beads in
  `.beads/beads.jsonl`, would we close that gap completely? Why or
  why not?
- If the answer is no, propose a concrete set of new beads (or
  revisions to existing beads) that would close the gap, and explain
  why each is needed.

Use **ultrathink**.
